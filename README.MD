## How to run locally

```shell
brew install ruby@3.0

echo 'export PATH="/usr/local/opt/ruby@3.0/bin:$PATH"' >> ~/.zshrc
# for apple silicon
# echo 'export PATH="/opt/homebrew/opt/ruby@3.0/bin:$PATH"' >> ~/.zshrc


source ~/.zshrc

bundle install

bundle exec jekyll serve

```
